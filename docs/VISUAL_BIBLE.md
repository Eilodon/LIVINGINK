# COLOR JELLY RUSH - VISUAL BIBLE (Eastern Mysticism x Grimdark)

## Core Identity
- Theme: Ritual ink, sacred geometry, lacquered stone, and ember-lit brass.
- Mood: Tension + serenity, like a temple under siege.
- Contrast: Warm gold/ember highlights against deep ink shadows.

## Color System
- Ink Base: `#0b0b0c` / `#151018` / `#241b27`
- Bone Light: `#f5eee3` / `#e1d6c8`
- Gold Accent: `#d1b06a`
- Ember Accent: `#c16c35`
- Jade Accent: `#2f8d6e`
- Blood Accent: `#b04a4a`
- Mist Text: `#9a8e9c`

Usage:
- UI surfaces: ink base gradients, gold trims.
- Actions: ember = primary, jade = confirm/success, blood = danger.
- Text: bone for primary, mist for secondary.

## Typography
- Display: Cinzel (ritual inscriptions).
- UI: Sora (clean, modern control legibility).
- Tracking: uppercase + wide spacing for headings and labels.

## Layout Language
- Panels feel carved: thick borders, soft bevel shadows.
- Cards float slightly: subtle lift on hover.
- Navigation uses sigil-like chips and pill labels.

## UI Components
- Primary CTA: gold-to-ember gradient, bold tracking.
- Secondary CTA: dark glass with gold trim.
- Tags/Pills: thin borders, muted ink base.
- Inputs: dark lacquer field with gold focus ring.

## Motion + VFX
- Use staggered reveals for menus (0.1s increments).
- In-game feedback: short pulses, ripples, and shockwaves.
- Avoid over-noise: keep particles anchored to gameplay meaning.

## Surreal Accent Layer
- Subtle radial gradients + faint diagonal grain pattern.
- Sacred geometry motifs appear around key goals (rings/core).

## Do / Dont
- Do: high-contrast silhouettes, ritual textures, cinematic glow.
- Dont: neon overload, flat monochrome surfaces, excessive UI chrome.
